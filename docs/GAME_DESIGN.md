# 🎮 Pixel Kingdom: Survival RPG

## Overview

Pixel Kingdom is a browser-based survival RPG game.

The player fights enemies, collects experience, upgrades abilities and defeats bosses.

---

## Platforms

- Desktop Browser
- Mobile Browser

---

## Main Features

- Player movement
- Enemy waves
- Combat system
- Weapons
- Skills
- Inventory
- Boss battles
- Achievements

---

## Technology Stack

Frontend:

- React
- TypeScript
- Phaser
- Tailwind CSS

Backend:

- Node.js
- Express
- PostgreSQL
- Prisma
